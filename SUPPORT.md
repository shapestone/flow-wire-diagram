# Support

## Usage Questions

Open a blank issue on the [GitHub issue tracker](https://github.com/shapestone/flow-wire-diagram/issues).

## Bug Reports

Use the bug report issue template when filing a bug. Include the input Markdown, the command you ran, and the actual vs. expected output.

## Security Issues

Do not open a public issue for security vulnerabilities. Instead, file a [private security advisory](https://github.com/shapestone/flow-wire-diagram/security/advisories/new). See [SECURITY.md](SECURITY.md) for the full policy.

## Support SLA

This is a solo-maintained project. There is no guaranteed response time or support SLA.
